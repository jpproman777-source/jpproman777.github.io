Introduccion

Bienvenidos a este ejemplo practico de un documento HTML, que utiliza una estructura semantica. Aqui exploraremos como organizar tu contenido de forma clara y ordenada, empleando solo parrafos y listas para que te familiariaces con las etiquetas basicas.

Pasos para Crear una Estructura semantica

En la siguiente lista ordenada se detallan los pasos fundamentales:

  1. Declarar el DOCTYPE y abrir la etiqueta HTML.
  2. Configurar el encabezado (head) con metas datos y el titulo.
  3. Organizar el contenido usando etiquetas semanticas como header, nav, main, article, section y footer.
  4. Utilizar parrafos para el conteido textual.
  5. Incorporar listas ordenadas y desordenadas para estructurar la informacion.
